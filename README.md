# Game Rest API
Django project that utilizes the rest framework in order to create a workable API for different video games. Only few added just to ensure working API, not intended for production.

# How to run
## Start server
In directory with manage.py file
- `python manage.py runserver`
## Connect to server
In browser
- `localhost:8000/`
## To 'use'
In postman or other software
- `http://localhost:8000/games/"title"`
- **"title" is case/space sensitive**
